# # 📱 TaskManagerPro - Gestor de Tareas

## 📋 Descripción

TaskManagerPro es una aplicación Android desarrollada en Java que permite gestionar tareas utilizando una base de datos SQLite local.

La aplicación implementa operaciones CRUD (Crear, Consultar, Actualizar y Eliminar) para administrar tareas de manera sencilla y eficiente.

---

## 🚀 Funcionalidades

* ✅ Agregar tareas
* 🔍 Buscar tareas por ID
* ✏️ Editar tareas existentes
* 🗑️ Eliminar tareas
* 📋 Mostrar todas las tareas registradas
* 💾 Almacenamiento local con SQLite

---

## 🛠 Tecnologías Utilizadas

* Java
* Android Studio
* SQLite
* XML
* Gradle
